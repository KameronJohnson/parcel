# Parcel

Calculates the price of sending a parcel based on its volume and weight.

## Setup Instructions

### Gems Used
```erb
sinatra
sinatra-contrib
```

### Install Bundler
```erb
$ gem install bundler
```

### Run Bundle
```erb
$ bundle
```

### Bug Reports
If user enters letters instead of numbers, the cost_to_send method will not work properly.
